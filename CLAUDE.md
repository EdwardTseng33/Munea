# Claude entrypoint for Munea

Before editing, follow `AGENTS.md` and check `docs/協作看板-雙AI分工.md` plus open GitHub pull requests.

Use your own branch and, when the checkout is dirty or shared, your own worktree. List the files you plan to change in the task or Draft PR. If another session is changing the same file, wait for its PR to merge, update from `origin/main`, and then continue. No JSON lock, lease, or lock-only PR is required.
