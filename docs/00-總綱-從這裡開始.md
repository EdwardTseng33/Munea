## 2. 核心定義（一眼看懂沐寧）
- **官方一句話定位（Edward 6/28 鎖定）**：「**會記得你，也在乎你在乎的人。**」靈魂＝懂得何時該講、何時該靜，讓人驚豔的感知與聊天互動。
- **是什麼**：住手機裡、會說話、記得你、出事會找人的「家人替身」。**子女買、長輩用**。智慧健康／照護、**非醫療軟體**。
- **核心畫面**：點開 → **全屏管家臉、會說話、會動、超大結束鈕**（像跟家人視訊）。畫面**像視訊通話、不放逐字稿/字幕當主畫面**；只顯示必要狀態（在聽／在說）與安全提示。字幕僅列為**未來給重聽長輩的可選輔助（預設關）**。→ 依 2026-06-29 S2S 產品框架校正（見 `STATUS.md`／`README.md`／`CURRENT-DEVELOPMENT-PLAN.md`），**取代舊「即時字幕為核心元素」的寫法**；SPEC §5/§6 仍有舊字幕寫法待同步（見 §4）。
- **三顆腦＋角色人格層＋一張臉**：反射腦(即時語音)＋管家腦(背景感知/記憶)＋守護腦(危機安全網)＋角色人格層(六角色、使用者命名、語氣/關係/表達)＋一張臉(FlashHead／2D fallback)。
- **產品架構說法（對外 · Edward 2026-07-02 定）**：三大核心服務模組 ＝ 🧠**記憶**（記得你）／🌤️**感知**（懂此刻）／💬**交互**（怎麼跟你互動；含語音／會動的臉／安全／角色人格／聽說互動調整）。對內展開為可換技術層＋指揮層，權威見 `聊聊-分層服務架構-2026-07-02`。
- **真正回話公式**：`回應 = 角色人格 + 使用者記憶 + 即時感知 + 當下對話 + 安全規則 + 語音表達限制`。
- **AI 模型現況（7/18 更新）**：即時語音主要為 Gemini Live；Butler／Guardian 的文件宣告、router 名稱與可執行 provider path 尚未完全一致，因此保持 `partial / blocked`，不得再把 `claude-sonnet-5` 的字串宣告寫成 production provider 已驗證。現況與缺口見 `PRODUCT-ALIGNMENT-REGISTER.md`。
- **付費模式現況（7/18 更新）**：**`Free / Plus / Pro`**。Free 綁定帳號後一次性 5 點；Plus 每月 100 點；Pro 每月 200 點；1 點約等於 1 分鐘 Voice + Avatar。權威見 `BILLING-CREDITS-ENTITLEMENT-v1.md`。
