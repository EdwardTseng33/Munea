## 1. 推進階段

### Stage 0 ✅（7/8 已完成 · 本機）
管線全通：聊聊語音鏈 × 即時對嘴 × 串流進瀏覽器；牆鐘對時、羽化貼合、位元率調校、插話重置。**這些工程資產全部移植到正式版。**

### Stage 1 · 即時化與基準（RunPod 4090 · 數天 · 驗證費個位數美元）
- Ditto 上 4090＋TensorRT 加速（官方有現成加速版；內部 6/26 離線驗證已過、缺的就是這步）
- 實測：720p 幀率、首幀延遲、單卡幾路、**語音腦每分鐘成本一併量回填**
- 產出：寧寧效果影片 × Tavus 對照組（免費 25 分鐘建寧寧人物）**盲測評分**——「離遠景多遠」從感覺變成數字

### Stage 2 · 生動度與畫質（1–2 週）
- **動態調參**：眨眼頻率、頭動幅度（長輩取向＝溫和穩定）、表情強度
- **情緒球 → 表情訊號**：長輩心情低落→寧寧神情放柔（遠景三家都沒有的差異化、Ditto 細粒度控制原生支援）
- 貼合羽化、色彩一致性（Stage 0 經驗移植）
- **畫質強化層 benchmark**：4090 對 512 臉區跑即時強化，幀率許可就上（嘴部細節再上一級）

### Stage 3 · 規模化（上線期）
- 按需開卡、沒人看就關（scale-to-zero）＋併發池化＋排隊
- 監控與成本迴歸：守住每分鐘 ≤ NT$0.5 信封
- 尖峰策略：自動降載（先降幀率再排隊、不斷線）

### Stage 4 · 長期研發選項（量力排程、不擋上線）
- **fork 自養**（Apache 權利在手）：上游停更也不受制於人
- 模型層評估：蒸餾/微調換更快更小、輸出解析度原生提升
- **端側化探索**：學 Duix 手機端架構（只學架構、不用它的授權）——臉跑在長輩手機上＝每分鐘成本趨近零，是成本的終局型態

## 3. 毛利帳（Edward 問：200 分鐘 Live Avatar、售價 NT$690 → 最終毛利率？）

**前提**：Apple 抽 15%（小型企業）→ 實收 NT$586.5；匯率 31；點數=分鐘、天生封頂。

### 用自架（拍板路線）
| 成本項（200 分全用滿＝最壞情況） | 金額 |
|---|---|
| 臉（4090 每分鐘 NT$0.36 / 併發 2 路則 0.18，含 20% 保溫攤提）| NT$43–86 |
| 語音腦（Gemini Live · 估、Stage 1 實測回填）| NT$40–90 |
| 其他變動（搜尋/推播/資料庫攤提）| NT$20–30 |
| **總變動成本** | **NT$103–206** |
| **毛利** | **NT$380–484** |
| **毛利率（最壞全用滿）** | **≈ 65–82%** |

- 典型用量（點數用 6 成）→ 毛利率 **75–85%**。✅ **690 這個價位健康。**

### 對照：若用租的（HeyGen LiveAvatar Lite 每分鐘 ~NT$3.1）
- 臉 200 分 = NT$620 → 加語音後總成本 **NT$680–740 > 實收 586.5** → **毛利率 -16%～-26%、賠錢**
- 損益兩平點：臉只能用到約 **155 分鐘**——重度戶必賠。
- → **再次驗證拍板：規模化只有自架能活**；租的只配當「效果對照組」與「極早期低用量過渡」，且要設分鐘上限。
