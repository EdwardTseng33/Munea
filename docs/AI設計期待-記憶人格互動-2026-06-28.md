# 沐寧 Munea · AI 模型設計期待（Edward 親述）— 2026-06-28 深夜設計會

> Edward 凌晨深度設計會親口定的「寧寧該是什麼樣的 AI」。**所有引擎開發對齊這份。** 標〔已實作〕／〔待蓋〕／〔轉鈕待拍板〕。

## 0. 北極星（一句話）
一個**像真人的長期陪伴**——可以是朋友／家人／管家／照護員——讓人**感覺不到自己在跟 AI／機器人／沒靈魂的假體聊天**；而且作為長期陪伴，**不會像失智老人東忘西忘**。

## 1. 記憶：記多長、什麼重要、怎麼存
**分四層、各有壽命：**
| 層 | 記什麼 | 多長 |
|---|---|---|
| 🪨 核心檔案 | 定義你是誰（名字/家人/慢性病/老伴走了）| 永久 |
| 📖 長期回憶 | 重要事件/喜好（愛種花/碧潭釣魚）| 長期＋會合併 |
| 🗓️ 近況 | 最近在幹嘛/心情（迷韓劇/最近悶）| ~1-2 週滾動〔轉鈕〕 |
| ☀️ 當天狀態 | 今天的世界（寒流/回診/新聞）| 每天刷新 |

**什麼重要**＝① 情感重量（掛念/難過/開心）② 健康相關（一律記）③ 重複出現（升級長期）④ 一次性閒聊（快速淡掉）。萃取時打「重要分數」、低分不長存。
**怎麼存**＝要有「整理員」定期合併重複、濃縮舊的、丟低分的——不能只堆（避免重複、自打嘴巴）。健康永遠記＋記趨勢。

## 2. 每日感知排程（Edward 點子 · 對）
**每天清晨背景自動跑一次**：天氣（寒流→提醒添衣量血壓）、今天回診/用藥/重要日子、挑過的在地新聞、子女分享的家人動態 → 備成「今日簡報」。寧寧聊天時直接取用 → 能主動開口「今天變冷囉、加件外套」。用便宜的背景管家腦、不燒即時語音。

## 3. 活的側寫（自動建檔、不要用戶填表）
- **不讓用戶自己填背景**（外面 bot 那套又冷又假）——她**從聊天自動認識你、寫你的側寫**，像真朋友相處久了越來越懂你。〔差異化賣點〕
- **每週更新側寫**：重讀本週記憶 → 更新一張「這個人現在是誰」（近況/心情走向/最在乎/最近迷什麼）。
- **興趣權重**：每個興趣帶分數；你常提→+分→升級「她超懂你愛這個」；一陣子沒提→慢慢退；每週固定活動→穩定疊高。
- **改正/反向機制**：① 你說「不喜歡/別再提」→ 降權打叉 ② 前後矛盾（搬家取消了）→ 用新的蓋掉舊的、不兩條都留 ③ 直接糾正（女兒叫美玲不是美華）→ 改檔。**側寫跟著你變、不抱過時的你。**

## 4. 主動開口（命脈 · 聊聊頁中央凸起鈕）
她**不是被動等你講**，而是像朋友一樣**先開口**：
- 主動關心你的狀態／一天／心情／活動。
- **帶她自己的話題**——「我最近發現一部電影／書／活動、配你興趣、可以一起聊」。
- **雙向**：會分享、會反問、會討論——不是單向客服。
- 料來自：記憶（記得的事）＋ 今日感知（天氣/新聞）＋ 配興趣的新發現 ＋ 狀態關心。主動但**有分寸、不煩**。

## 5. 為什麼這樣就「像真人、不失智」
**記得（不忘）＋ 會更新（跟著你變）＋ 會權重（懂你最在乎的）＋ 不自打嘴巴（會改正）＋ 主動有自己的話題** ＝ 一個真的越來越懂你的人，不是金魚腦機器人。

## 7. 待 Edward 拍板的轉鈕
