# 沐寧「臉」畫質優化路線調研（2026-07-11 · 霍爾）

> 題目：在「自架＋即時（非 demo）＋顧毛利」前提下，找畫質明顯優於 512、每人成本可接受的最佳路線。
> 方法：三根槓桿逐一查證，每條主張標信心程度（**實測**＝我們自己或官方跑出的硬數字／**推測**＝用同代硬體公開 benchmark 類比、非 FlashHead 專測／**查不到**＝誠實列缺口）。
> 配套：`avatar-模型優化深度調研-2026-07-10.md`（引擎格局全貌）、`FlashHead官方對照體檢-2026-07-11.md`（我方管線 vs 官方逐行對照）、`聊聊臉-顯示與影音-定案手冊.md`（現行架構單一真相）、`1024與demo影片-技術評估-2026-07-11.md`（1024 已有結論）。

**槓桿 3 結論：**
1. **L40S、A100 在本工作負載下被 4090/5090 支配（同速或加速有限、價格更高），直接排除。**
2. **RTX 5090 是唯一「加速倍率夠、價格漲幅可控」的換卡選項**——@768 用 5090 能把餘裕從 4090 的 18.7%（偏險）拉到 45%（安全），代價是每人時薪從 NT$9.8 漲到 NT$17.4（+78%）。仍遠低於產品文件 NT$0.5/分鐘（≈NT$30/hr）的毛利天花板，**空間夠**。
3. **H100/1024 這條路成本暴增 5-6 倍，且 1024 本身就不是本代模型的原生解析度**——與 `1024與demo影片-技術評估` 文件既定結論一致：**1024 只走 demo 離線，不進即時候選**，此處不重複判死、只是再次印證。
4. **L4 是「省更多但更糊」的反方向**，不符合本次「畫質提升」題目，不列入排名。
5. 換卡選項全部標「推測」——**任何一張新卡在拍板前，先花 NT$30-100 租 1-2 小時實測 FlashHead 真實 chunk 時間**，不能只信 SDXL 類比數字就下單，這是燒錢 Gate 的硬要求。
