> 原則：**測試環境＝正式服務的縮小版**。測完不丟——同一套流程、同一套安全規矩、同一套監控，直接長成上線版。

## 1. 架構（測試即正式縮小版）

```
長輩手機／瀏覽器
   │  加密連線（https/wss）＋ 短效通行碼門禁（同 demo-cloud 發碼窗口模式）
   ▼
雲端 4090 主機（RunPod · 按需開、沒人用就關）
   ├─ 臉引擎：Ditto（TensorRT 即時版）→ 即時影像（WebRTC）
   ├─ 語音橋：聊聊 Gemini 語音鏈（今日已驗證那套、原樣搬上去）
   ├─ 門禁：發碼窗口（30 分鐘單次通行碼、正式鑰匙不出後台）
   └─ 監控記錄：幀率／首幀／嘴音同步差／GPU 使用率／每分鐘成本
本機（Windows）
   └─ 開卡關卡控制器（手動版先行 → 自動 scale-to-zero）
```

- 影像穿牆（手機行動網路）：備 TURN 中繼（WebRTC 標配），手機實測必過。
- 人物：**寧寧官方形象**（人物識別鐵律、不可換臉）。

## 2. 比照正式的檢核單（測試也要全過）

| 面向 | 檢核 |
|---|---|
| 🔐 安全 | 鑰匙只放環境變數不進程式庫；連線必帶通行碼；只開必要的門；上線前沙利曼信任關卡走一次 |
| 💰 成本 | 開卡即計時；**測試預算上限 US$15、到頂自動關卡**；所有花費記錄回填毛利表 |
| 🔁 可靠 | 斷線重連、插話打斷、連續 10 分鐘通話不斷、關卡→再開卡的冷啟動時間實測 |
| 🎭 品質 | **Edward 自然度四項**逐項 1–5 分（表情肌肉/眉毛、規則眨眼含睫毛、嘴幅與嘴角、頭部律動）＋ Tavus 免費額度建寧寧對照組盲測 |
| ⚡ 效能 | 720p 幀率、首幀 <1 秒、嘴音同步 <120ms、單卡 1／2／3 路壓測 |
| 📱 真機 | iPhone Safari 行動網路實測（不只電腦區網）|

## 3. 時程（對應七天劇本 D1–D4）

| 天 | 內容 | 產出 |
|---|---|---|
| D1 | 開卡、裝 Ditto＋TensorRT、離線出第一段寧寧影片 | 四項評分 v0（離線版） |
| D2 | 即時化：串流管線接上（本機資產移植）、語音橋上雲 | 手機能打開的測試連結 |
| D3 | 品質調參（眨眼節奏/頭動幅度/表情強度）＋對照組盲測 | 四項評分 v1 ＋盲測報告 |
| D4 | 壓測（幾路）＋冷啟動＋成本結算 | 實測毛利表、上線建議書 |

滑期保險：任一天卡死 → 按七天劇本保險絲 B/C，不影響上線承諾。

## 4. Edward 一次性動作（等蘇菲說 GO 再做、約 5 分鐘）

1. 到 `runpod.io` 註冊帳號、儲值 **US$25**（測試預算 15＋緩衝）
2. 帳號選單 → **Settings → API Keys** → 建立一把 → 貼給蘇菲
3. 之後開卡、關卡、看成本都蘇菲來，**每天跟你報花了多少**
